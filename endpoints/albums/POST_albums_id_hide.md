# POST /albums/#{id}/hide 
***
`/albums/#{id}/hide`

### Description
***
Hide an album from the entire community's discover (for admins only)


### Parameters
***

|header| description| type |required? |default|
|:---------|:--------------|:----------:|:------------:|:------------:|
|**id**| the id of the album to share|integer|x||
|**parameter**| **description**| **type** |**required?** |**default**|


### Response
***
200 if success
