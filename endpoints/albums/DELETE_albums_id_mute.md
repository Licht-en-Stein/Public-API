# DELETE /albums/#{id}/mute 
***
`/albums/#{id}/mute`

### Description
***
Unmute an album from a specific user's discovery.


### Parameters
***

|header| description| type |required? |default|
|:---------|:--------------|:----------:|:------------:|:------------:|
|**id**| the id of the album to share|integer|x||
|**parameter**| **description**| **type** |**required?** |**default**|


### Response
***
200 if success
