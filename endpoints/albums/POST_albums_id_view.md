# POST /albums/#{id}/view 
***
`/albums/#{id}/view`

### Description
***
add an extra "view"(visit) to the user's record of interaction with the album.


### Parameters
***

|header| description| type |required? |default|
|:---------|:--------------|:----------:|:------------:|:------------:|
|**id**| the id of the album|integer|x||
|**parameter**| **description**| **type** |**required?** |**default**|


### Response
***
200 if success
